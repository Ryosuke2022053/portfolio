＜趣味＞

私の趣味は旅行です。日本国内のみならず海外にも強い興味があります。
今までも色々なところに行きましたが、観光することが好きなのではなく、目的地までどのようなスケジュールで行くか、
どの手段で行くかなどの計画を立てることが好きで得意だと思います。

＜インド＞

2022年の5月から３ヶ月間オンライン授業とリモートワークをしながらインドを旅しました。
インドという国はコロナの影響を全く感じられず、コロナ禍でもしっかり経済成長をしていたということを旅を通して強く感じています。
特にシャッター街や休業中のレストランなどは一切なく、中流家庭以上のインド人の購買力は日本人の僕よりあるなと感じます。
ただ街のあっちこっちにスラム街があり、貧富の格差もかなり感じます。

LinkedInにてインド旅のブログを投稿しています。下記のURLから閲覧することができます。

＜業務経歴＞

アンダーズ株式会社での長期インターンシップ経験
自身のスキル向上や社会経験のために、2021年9月からエンジニアインターンに取り組みました。
インターン先では主に上流工程を経験させてもらいました。インターン先の会社では下流工程はインド在住のインド人に任せていました。
そのため、コミュニケーションにはとても苦労しました。インターンシップを通して、相手への簡潔に情報を伝えることや相手が欲しい情報は何なのかを常に考えることの重要さを学びました。
